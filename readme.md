# Highlight New Albums for YouTube Music

Highlights albums on the YouTube Music New Releases page, so you can ignore the filler

## Features

-   Puts a green background and border around all Albums listed on the New Releases page
-   Works in browser or on the PWA (desktop installed App)
-   Only can be activated if you are on https://music.youtube.com/new_releases/albums

Started as a bookmarklet I wrote: the unminified version is in the root of this repo
